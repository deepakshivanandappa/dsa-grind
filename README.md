# 🧠 DSA Grind - Interview Prep Tracker

A single HTML file that combines curated list of LeetCode problems into one searchable, filterable interview prep tracker.

**No account. No server. No cost. Ever.**
Your progress is saved in your own browser - completely private, works offline.

🔗 **[Live → deepakshivanandappa.github.io/dsa-grind](https://deepakshivanandappa.github.io/dsa-grind/)** 

---
### Features
- ✅ Completion checkbox per problem
- 🔢 Practice counter - track how many times you've solved it
- 📝 Notes per problem - approach, time/space complexity
- 📊 Progress bar - overall and per-category
- 🔍 Search + filter by difficulty, source, completion status
- ⬆⬇ Export/Import JSON backup - move between devices
- 🌙 Dark theme, works offline

---

## How progress is saved

Your data lives in your **browser's localStorage** - not on any server, not on GitHub. This means:

- ✅ Close the tab, come back tomorrow - progress is exactly where you left it
- ✅ Works offline after the first load
- ⚠️ Switching browsers or computers = no data (use the Export button to back up)
- ⚠️ Clearing browser data wipes progress (export first!)
